# Ecuador Company Intelligence MVP

Portal estático de inteligencia comercial para empresas de Ecuador, construido con datos públicos de la Superintendencia de Compañías, Valores y Seguros (SCVS), ranking 2024.

## Contenido

- Dashboard general
- Ranking filtrable de Top 1.000 empresas
- Ficha rápida por empresa
- Score inicial de oportunidad comercial
- Datos JSON en `data/`

## Fuente

SCVS Ecuador — recursos públicos de ranking empresarial 2024.
